# Railway
An Script to host on railway.app ~

# Add in your env

- REPO_URL
- START_CMD


# Deploy 

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/new/template/RXeiAo?referralCode=GTp-zc)


## Through Railway CLI
```sh
npm i -g @railway/cli
railway login
railway init
railway link
railway up --detach
```
